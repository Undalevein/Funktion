## About Funktion

Funktion is a functional program language...

- Functional Language
- Generators
- Math Audience
- Graph [Library Stuff]
- Ranges (Ex: `0 .. 5` or `0 ..` `0:10`) (Ex: ``)
  - `0..(+2)..10`
  - `0..10 (.2)`
  - `0..10 t.2t`
  - `0..`
  - `0.. t.2t`
- f(x) = 2x
- Slices
  - list = [1,2,3]; list[2:]
  - [f(x)]: `.3...9`
- Strings
  - String of x
    - char(x) = 'a'..'z'
- Syntax
  - Keywords:
    - print
